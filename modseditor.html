jQuery.modseditor

View the Project on GitHub UNC-Libraries/jquery.modseditor

A browser based, schema driven XML editor developed by the University of North Carolina at Chapel Hill Libraries. It was expressly developed for editing MODS documents, but is usable for many other XML schemas as well.

This project was developed as a part of the Carolina Digital Repository's administrative interface, but is a stand alone jQuery UI widget. Source code for the CDR also is available via github.
